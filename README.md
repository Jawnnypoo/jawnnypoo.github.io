# jawnnypoo.github.io
My personal site showcasing some of my work.

This site is derived from [Polymer Starter Kit Lite](https://github.com/polymerelements/polymer-starter-kit/) and it showcases most of my personal work and projects. Polymer is awesome, and you should give it a go. 

## Using the code here

Feel free to fork the project, though keep in mind the code here is not vulcanized or optimized (to keep it readable) so I would recommend grabbing the full Polymer Starter Kit for production sites.
